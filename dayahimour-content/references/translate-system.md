| الحقل | القاعدة |
|-------|---------|
| `title` | ترجمة طبيعية بـ Title Case (الكلمات الرئيسية كبيرة). أسماء النماذج والشركات كما هي |

| `draft` | كما هو |
| `tier` | احتفظ بالقيمة كما هي (`flagship` / `standard` / `budget`) — لا تُترجم |

### خريطة التصنيفات (case-sensitive — أحرف صغيرة فقط)
```
أخبار    → news       (لا News)
تحليل    → analysis   (لا Analysis)
نماذج    → models     (لا Models)
أدوات    → tools      (لا Tools)
شركات    → companies  (لا Companies)
مستقبل   → future     (لا Future)
صور      → images     (لا Images)
فيديو    → video      (لا Video)
مقارنات  → comparisons (لا Comparisons)
```

> ⚠️ **خطأ شائع فادح**: المخطط (schema) يقبل **أحرف صغيرة فقط**. كتابة `"Models"` بدلاً من `"models"` تُسبب **فشل البناء كامل**.
> ⚠️ لا توجد قيمة `"breaking"` — `importance` يترجم من `"عادي"` → `"normal"` و `"بارز"` → `"prominent"` فقط.

## قواعد ترجمة المحتوى

### ما يُترجم
- النصوص العربية كاملاً بأسلوب صحفي احترافي (The Verge / Wired)
- التواريخ العربية في النص: "أبريل 2026" → "April 2026"
- تسميات الفئات في أشرطة Benchmark:
  - رياضيات → Math
  - علوم → Science
  - برمجة → Coding
  - متعدد الوسائط → Multimodal
  - معرفة عامة → General Knowledge
  - استنتاج → Reasoning
- "المعايير المرجعية" → "Benchmark Results" أو "Performance Metrics"
- "المعايير المرجعية الرئيسية" → "Key Benchmark Results"

### ما لا يُترجم أبداً
- أسماء النماذج: GPT-5, Claude Opus 4.6, Gemini, MiMo-V2.5-Pro
- أسماء الشركات: OpenAI, Anthropic, Meta, Google, xAI, DeepSeek
- أسماء المعايير: AIME, GPQA Diamond, SWE-bench, ARC-AGI-2, MMLU, HumanEval, MMMU, LiveCodeBench
- مصطلحات تقنية: MoE, fine-tuning, chain of thought, overfitting, RLHF
- أسماء المنصات: GitHub, Reddit, r/LocalLLaMA, LM Arena
- الأرقام والإحصاءات
- المحتوى داخل `style="..."` attributes
- المحتوى داخل code blocks (```)
- قيمة حقل `tier` في الـ frontmatter

### أشرطة HTML (حرج جداً)
- احتفظ بكل CSS والألوان والتدرجات والعروض والهيكل كما هو بالضبط
- **ترجم فقط** النصوص العربية المرئية (تسميات الفئات، عنوان القسم، المفتاح التوضيحي)
- **لا تُعدّل**: `width: XX.X%`, ألوان الخلفية، التدرجات، الأرقام

## قواعد صارمة

1. **لا تُضف محتوى** غير موجود في الأصل
2. **لا تحذف محتوى** من الأصل
3. **لا تُغير هيكل المقالة** أو ترتيب العناوين
4. **لا "you/your"** — حافظ على الأسلوب الصحفي بضمير الغائب
5. **لا إيموجي** إلا إذا كانت موجودة في الأصل
6. **الطول**: النسخة الإنجليزية بنفس طول الأصل تقريباً
7. **اسم الملف**: يبقى كما هو (نفس الـ slug)
8. **لا مصطلحات زمنية مؤقتة** في الترجمة: ترجم التاريخ العربي إلى التاريخ الإنجليزي المكافئ
9. **الاسم الرسمي**: احتفظ بالاسم الرسمي للنموذج/المنتج كما هو
