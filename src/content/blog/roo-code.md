## ما هو Roo Code؟

Roo Code هو **إضافة لـ VS Code** تنشئ فريقاً من وكلاء AI المتخصصين:

- **وكيل الواجهة الأمامية** — متخصص في HTML/CSS/JavaScript
- **وكيل الواجهة الخلفية** — متخصص في APIs وقواعد البيانات
- **وكيل الاختبارات** — متخصص في كتابة الاختبارات
- **وكيل المراجعة** — متخصص في مراجعة الكود

### كيف يعمل الفريق؟

```
أنت: "ابنِ تطبيق متجر إلكتروني"

Roo Code يوزع المهام:
1. ✅ وكيل الواجهة الأمامية: يبني صفحات المتصفح
2. ✅ وكيل الواجهة الخلفية: يبني APIs
3. ✅ وكيل قاعدة البيانات: يصمم الجداول
4. ✅ وكيل الاختبارات: يكتب الاختبارات
5. ✅ وكيل المراجعة: يراجع كل الكود
```
