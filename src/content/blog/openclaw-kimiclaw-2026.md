## ظهور فئة "Claw" في أدوات الذكاء الاصطناعي الشخصية

في يناير 2026، ظهر مشروع **"Clawdbot"** في نشرات المطورين التقنية — أداة تتيح تشغيل وكيل AI يتواصل عبر واتساب وينجز مهام حقيقية. خلال أسبوعين، وصل المشروع لـ10,000 نجمة على GitHub.

تغيّر الاسم لاحقاً إلى **Moltbot** ثم **OpenClaw**. وبنهاية فبراير 2026، تجاوز المشروع 100,000 نجمة على GitHub.

في 15 فبراير، أطلقت **Moonshot AI الصينية KimiClaw** — نسخة سحابية تعمل عبر المتصفح دون إعداد تقني.

يشهد المشهد الآن اتجاهين: من يفضل التحكم الكامل (OpenClaw) ومن يفضل السهولة (KimiClaw).

## OpenClaw: الوكيل المحلي المستضيف

### الفكرة الأساسية

OpenClaw يعمل كعملية خلفية مستمرة (persistent background process) على الجهاز، ويستقبل الطلبات عبر تطبيقات المراسلة المختلفة.

### البنية التقنية

```
[واتساب/تليجرام/ديسكورد]
         ↓
[Gateway (Node.js - Port 18789)]
         ↓
[نظام التوجيه]
         ↓ ↙ ↓ ↙ ↓
[LLM]  [أدوات]  [ذاكرة]
         ↓
[تنفيذ + مراقبة النتيجة]
         ↓
[الرد عبر التطبيق]
```

### المهارات المدمجة (+100 في أبريل 2026)

**مهارات التواصل:**
- إرسال/قراءة واتساب ورسائل SMS
- إدارة البريد الإلكتروني (Gmail/Outlook)
- نشر على تويتر/X وLinkedIn

**مهارات المعلومات:**
- بحث الويب متعدد المحركات
- قراءة مواقع ومقالات وPDFs
- استخراج بيانات منظمة (scraping)

**مهارات التنظيم:**
- إضافة مواعيد لـGoogle Calendar
- إنشاء وتعديل ملفات Google Docs/Sheets
- إدارة قوائم المهام في Notion وTodoist

**مهارات التطوير:**
- تشغيل كود Python/JavaScript
- استدعاء APIs خارجية
- قراءة وتعديل ملفات النظام

**مهارات المتصفح:**
- فتح مواقع والتنقل فيها
- ملء نماذج
- حجز مواعيد وتذاكر

## تثبيت OpenClaw

### المتطلبات المطلوبة

| المتطلب | المواصفات المطلوبة |
|---------|---------------------|
| نظام التشغيل | Mac / Windows / Linux |
| الذاكرة RAM | 8GB كحد أدنى، 16GB موصى |
| Node.js | v20 أو أحدث |
| API Key | Claude أو GPT أو Gemini |
| اتصال الإنترنت | مستمر (للمهام الخارجية) |

### خطوات التثبيت

```bash
# تنزيل المشروع
git clone https://github.com/OpenClaw/openclaw.git
cd openclaw

# تثبيت المتطلبات
npm install

# إعداد ملف الإعدادات
cp .env.example .env
```

إعداد ملف .env:
```env
# اختيار مزود النموذج
ANTHROPIC_API_KEY=sk-ant-xxxxx
# أو
OPENAI_API_KEY=sk-xxxxx
# أو
GOOGLE_API_KEY=xxxxx

LLM_PROVIDER=anthropic
LLM_MODEL=claude-sonnet-4-6

MESSAGING_PLATFORM=whatsapp  # أو telegram أو discord

MEMORY_TYPE=local  # local أو postgres
```

```bash
# تشغيل الوكيل
npm run start
```

### إضافة مهارات مخصصة

```javascript
// src/skills/custom/reminder.js
module.exports = {
  name: "set_reminder",
  description: "يضع تذكيراً في وقت محدد",
  parameters: {
    message: "نص التذكير",
    time: "الوقت (مثال: بعد ساعة، الساعة 3 مساء)"
  },
  execute: async ({ message, time }) => {
    const parsedTime = parseArabicTime(time);
    await scheduleReminder(message, parsedTime);
    return `تم إعداد التذكير: "${message}" في ${parsedTime}`;
  }
};
```

## KimiClaw: الحل السحابي الجاهز

### الفكرة الأساسية

KimiClaw يعمل عبر المتصفح دون أي إعداد تقني. البيانات تُخزَّن على خوادم Moonshot AI.

### المميزات التقنية

**Kimi K2.5 — النموذج المدمج:**
متخصص في التفكير متعدد الخطوات بأداء تنافسي في معايير البرمجة والاستدلال.

**5000+ مهارة من ClawHub:**
- حجز مواعيد المستشفيات في مدن صينية متعددة
- تحليل فيديوهات Bilibili وYouTube
- متابعة أسعار الصرف والتنبيه عند تجاوز حد معين
- مهارات متخصصة في الأسواق الخليجية

**40GB تخزين سحابي:**
مع إمكانية الحذف الكامل.

### خطوات البدء

1. فتح browser.kimi.ai
2. التسجيل بحساب Google أو البريد الإلكتروني
3. تفعيل وضع Agent Mode
4. اختيار المهارات من ClawHub
5. بدء المحادثة

## مقارنة تفصيلية: OpenClaw مقابل KimiClaw

| المعيار | OpenClaw | KimiClaw |
|---------|---------|---------|
| **نوع الاستضافة** | ذاتية (على الجهاز) | سحابية (Moonshot) |
| **وقت الإعداد** | 30-60 دقيقة | 3 دقائق |
| **الخصوصية** | عالية (100% محلي) | متوسطة |
| **التكلفة** | مجاني + تكلفة API | مجاني (محدود) / مدفوع |
| **عدد المهارات** | 100+ (قابلة للتوسع) | 5000+ (جاهزة) |
| **النموذج** | أي نموذج يُختار | Kimi K2.5 (مدمج) |
| **تكامل المراسلة** | واتساب/تليجرام/ديسكورد | عبر المتصفح |
| **الذاكرة** | دائمة (محلية) | دائمة (سحابية) |
| **التوافر** | يعتمد على الجهاز | 24/7 سحابي |
| **تطوير مخصص** | كامل (كود مفتوح) | محدود |
| **متطلبات تقنية** | متوسطة (Node.js) | صفر |

## مقارنة عملية: نفس المهمة، نهجان مختلفان

**المهمة:** "متابعة سعر سهم ARAMCO يومياً في 9 صباحاً مع ملخص التغيير"

**OpenClaw:**
```javascript
// skill: stock_tracker.js
module.exports = {
  schedule: "0 9 * * 0-4",  // كل يوم عمل الساعة 9
  execute: async () => {
    const price = await fetchStockPrice("2222.SR");
    const change = calculateChange(price, previousPrice);
    await sendWhatsApp(
      `أرامكو: ${price} ريال (${change}%)\n` +
      `الحد الأدنى اليوم: ${price.low}\n` +
      `الحد الأعلى: ${price.high}`
    );
  }
}
```
يعمل تلقائياً يومياً حتى الإيقاف اليدوي.

**KimiClaw:**
كتابة الطلب في المحادثة → تفعيل مهمة مجدولة → إرسال إشعار في المتصفح.

الفرق الأساسي: OpenClaw يُرسل عبر واتساب دون فتح المتصفح. KimiClaw يتطلب وجود إشعارات المتصفح مفعّلة.

## حالات الاستخدام

### OpenClaw مناسب لـ:

**الوكيل البريدي الكامل:**
مراسلة العملاء الجدد تلقائياً وإضافتهم لـCRM.

**تتبع المنصات المتعددة:**
متابعة ذكر اسم الشركة في تويتر أو Reddit.

**الاستضافة المحلية للبيانات الحساسة:**
للشركات التي لا ترفع بياناتها للسحابة.

### KimiClaw مناسب لـ:

**المهام الفورية بدون إعداد:**
تحليل الكود مباشرة.

**استخدام مهارات المجتمع الجاهزة:**
مثل "تحليل PDF العقود" أو "ترجمة وتقارير".

**التعاون الفوري:**
مشاركة جلسة مع زميل بنفس الوكيل.

## المشاريع المشتقة

### PicoClaw — للأجهزة الضعيفة

مُصمَّم لأجهزة ARM الرخيصة (Raspberry Pi وOrange Pi). يستخدم Phi-3.5 Mini — أخف لكن أقل قدرة.

### JoonClaw (قريباً)

مشروع قادم من Joon AI يدعي تفوق نموذج K2.5 على منافسيه. لم يصدر رسمياً حتى أبريل 2026.

### NanoClaw — الأمان المتقدم

نسخة معدَّلة تعمل داخل Docker معزول بصلاحيات صارمة — للشركات التي تحتاج مستويات أمان عالية.
