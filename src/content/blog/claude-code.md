## ما هو Claude Code؟

Claude Code هو **واجهة مخصصة للمبرمجين** من نموذج Claude من Anthropic. يتميز بـ:

- **فهم عميق للكود** — يحلل مشاريع كاملة
- **شرح واضح** — يشرح المفاهيم المعقدة ببساطة
- **تصميم الحلول** — يساعد في تصميم البنية المعمارية
- **مراجعة الكود** — يجد الأخطاء ويقترح تحسينات
