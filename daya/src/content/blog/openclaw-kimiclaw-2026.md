## مقدمة: لماذا "Claw" أصبحت الكلمة الأكثر تداولاً في عالم AI الشخصي؟

في يناير 2026، بدأ يظهر في نشرات المطورين التقنية اسم **"Clawdbot"** — مشروع صغير يُتيح تشغيل وكيل AI يتحدث معك عبر واتساب ويُنجز مهام حقيقية. بعد أسبوعين، وصل المشروع لـ10,000 نجمة على GitHub.

غيّروا الاسم لـ**Moltbot**. ثم لـ**OpenClaw**.

**بنهاية فبراير 2026: أكثر من 100,000 نجمة على GitHub**. وصفه Jen-Hsun Huang (رئيس Nvidia) بأنه "ChatGPT المقبل" في مؤتمر GTC.

ثم في 15 فبراير، أطلقت **Moonshot AI الصينية KimiClaw** — نسخة سحابية تعمل بالمتصفح دون أي إعداد تقني.

**الآن في أبريل 2026:** المشهد انقسم بين معسكرَين: من يُريد التحكم الكامل (OpenClaw) ومن يُريد السهولة المطلقة (KimiClaw). هذا الدليل يُساعدك على الاختيار.

## ما هو OpenClaw؟ — الوكيل الذي يعيش على جهازك

### الفكرة الجوهرية

OpenClaw ليس تطبيقاً تُفتحه وتُغلقه — بل **عملية خلفية مستمرة (persistent background process)** تعمل على جهازك وتنتظر طلباتك في أي وقت، عبر أي تطبيق مراسلة تختاره.

**تخيّل:** وكيل شخصي دائم الحضور، يعمل 24 ساعة، يتذكر كل ما ناقشتموه، ويستطيع تنفيذ مهام حقيقية على الإنترنت وجهازك — كل هذا على خوادمك الخاصة دون أن تُرسل بياناتك لأحد.

### كيف يعمل: البنية التقنية

```
[أنت عبر واتساب/تليجرام]
         ↓
[Gateway (Node.js - Port 18789)]
         ↓
[نظام التوجيه - يحدد ما إذا كان الطلب يحتاج أداة أم لا]
         ↓ ↙ ↓ ↙ ↓
[LLM]  [أدوات]  [ذاكرة]
         ↓
[تنفيذ + مراقبة النتيجة]
         ↓
[ردك عبر نفس التطبيق]
```

### قائمة المهارات المدمجة (+100 في أبريل 2026)

**مهارات التواصل:**
- إرسال/قراءة واتساب ورسائل SMS
- إدارة البريد الإلكتروني (Gmail/Outlook)
- نشر على تويتر/X وLinkedIn

**مهارات المعلومات:**
- بحث الويب متعدد المحركات
- قراءة مواقع ومقالات وPDFs
- استخراج بيانات منظمة (scraping)

**مهارات التنظيم:**
- إضافة مواعيد لـGoogle Calendar
- إنشاء وتعديل ملفات Google Docs/Sheets
- إدارة قوائم المهام في Notion وTodoist

**مهارات التطوير:**
- تشغيل كود Python/JavaScript
- استدعاء APIs خارجية
- قراءة وتعديل ملفات النظام (بإذن صريح)

**مهارات المتصفح:**
- فتح مواقع والتنقل فيها
- ملء نماذج (forms)
- حجز مواعيد وتذاكر

## تثبيت OpenClaw: خطوة بخطوة

### المتطلبات

| المتطلب | المواصفات الموصى بها |
|---------|---------------------|
| نظام التشغيل | Mac / Windows / Linux |
| الذاكرة RAM | 8GB كحد أدنى، 16GB موصى |
| Node.js | v20 أو أحدث |
| API Key | Claude أو GPT أو Gemini |
| اتصال الإنترنت | مستمر (للمهارات الخارجية) |

### التثبيت السريع

```bash
# الخطوة 1: تنزيل OpenClaw
git clone https://github.com/OpenClaw/openclaw.git
cd openclaw

# الخطوة 2: تثبيت المتطلبات
npm install

# الخطوة 3: إعداد الإعدادات
cp .env.example .env
```

**تحرير ملف .env:**
```env
# النموذج الذي تريد استخدامه (اختر واحداً)
ANTHROPIC_API_KEY=sk-ant-xxxxx
# أو
OPENAI_API_KEY=sk-xxxxx
# أو
GOOGLE_API_KEY=xxxxx

# اختيار النموذج
LLM_PROVIDER=anthropic
LLM_MODEL=claude-sonnet-4-6

# منصة المراسلة
MESSAGING_PLATFORM=whatsapp  # أو telegram أو discord

# الذاكرة
MEMORY_TYPE=local  # local أو postgres
```

```bash
# الخطوة 4: تشغيل الوكيل
npm run start

# ستظهر رسالة:
# ✅ OpenClaw running on port 18789
# 📱 Connect WhatsApp: scan QR code below
# [QR Code]
```

**الخطوة 5:** امسح رمز QR بواتساب → أرسل "مرحباً" → الوكيل يرد ويبدأ العمل.

### إضافة مهارات مخصصة

```javascript
// src/skills/custom/reminder.js
module.exports = {
  name: "set_reminder",
  description: "يضع تذكيراً في وقت محدد",
  parameters: {
    message: "نص التذكير",
    time: "الوقت (مثال: بعد ساعة، الساعة 3 مساء)"
  },
  execute: async ({ message, time }) => {
    // منطق إعداد التذكير
    const parsedTime = parseArabicTime(time);
    await scheduleReminder(message, parsedTime);
    return `✅ تم إعداد التذكير: "${message}" في ${parsedTime}`;
  }
};
```

## ما هو KimiClaw؟ — السهولة أولاً

### الفكرة الجوهرية

KimiClaw فلسفته العكسية تماماً لـOpenClaw: **لا تحتاج تثبيت، لا إعداد، لا server، لا كود**. افتح متصفح Kimi في browser.kimi.ai، وسجّل الدخول، وابدأ.

**الثمن:** بياناتك على خوادم Moonshot AI (الصين) لا على جهازك.

### المميزات التقنية لـKimiClaw

**Kimi K2.5 — النموذج المدمج:**
النموذج المُطوَّر من Moonshot يُقدَّم كـ"متخصص في التفكير المتعدد الخطوات" وأثبت أداءً تنافسياً في معايير البرمجة والاستدلال.

**5000+ مهارة من ClawHub:**
المجتمع طوّر مهارات متخصصة بشكل لافت:
- مهارة متخصصة لحجز مواعيد المستشفيات في 15 مدينة صينية
- مهارة لتحليل الفيديوهات على Bilibili وYouTube معاً
- مهارة لمتابعة سعر صرف العملات وتنبيهك عند تجاوز حد معين
- مهارات خاصة بالسوق الخليجي تطورها مجتمع عربي ناشئ

**40GB تخزين سحابي:**
الملفات والوثائق التي تُرفع تُخزَّن في سحابة Moonshot — مع ضمانات الخصوصية وإمكانية الحذف الكامل.

### كيفية البدء مع KimiClaw (3 دقائق فعلاً)

1. افتح **browser.kimi.ai**
2. سجّل بحساب Google أو بريدك
3. انقر "Enable Agent Mode"
4. اختر مهاراتك من ClawHub
5. ابدأ المحادثة

لا يوجد خطوة 6.

## المقارنة الشاملة: OpenClaw vs KimiClaw

### الجدول المرجعي الكامل

| المعيار | OpenClaw | KimiClaw |
|---------|---------|---------|
| **نوع الاستضافة** | ذاتية (على جهازك) | سحابية (Moonshot) |
| **وقت الإعداد** | 30-60 دقيقة | 3 دقائق |
| **الخصوصية** | ✅ ممتازة (100% محلي) | ⚠️ جيدة (BYOC متاح) |
| **التكلفة** | مجاني + تكلفة API | مجاني (محدود) / مدفوع |
| **عدد المهارات** | 100+ (قابلة للتوسع) | 5000+ (جاهزة) |
| **النموذج** | أي نموذج تختاره | Kimi K2.5 (مدمج) |
| **تكامل المراسلة** | واتساب/تليجرام/ديسكورد | عبر المتصفح فقط |
| **الذاكرة** | دائمة (محلية) | دائمة (سحابية) |
| **التوافر** | يعتمد على جهازك | 24/7 سحابي |
| **السرعة** | يعتمد على جهازك + API | سريعة عادةً |
| **تطوير مخصص** | ✅ كامل (كود مفتوح) | محدودة |
| **متطلبات تقنية** | متوسطة (Node.js) | صفر |

---

### المقارنة العملية: نفس المهمة، نتيجتان

**المهمة:** "تابع سعر سهم ARAMCO كل يوم في 9 صباحاً وأرسل لي ملخصاً مع التغيير"

**OpenClaw:**
```javascript
// skill: stock_tracker.js
module.exports = {
  schedule: "0 9 * * 0-4",  // كل يوم عمل الساعة 9
  execute: async () => {
    const price = await fetchStockPrice("2222.SR");  // رمز أرامكو
    const change = calculateChange(price, previousPrice);
    await sendWhatsApp(
      `📊 أرامكو: ${price} ريال (${change}%)\n` +
      `الحد الأدنى اليوم: ${price.low}\n` +
      `الحد الأعلى: ${price.high}`
    );
  }
}
```

تكتب هذا مرة واحدة → يعمل كل يوم تلقائياً على جهازك حتى تُوقفه.

**KimiClaw:**
تكتب في المحادثة: "كل يوم عمل الساعة 9 صباحاً، أعطني سعر سهم أرامكو مع نسبة التغيير" → يُفعّل مهمة مجدولة → يُرسل إشعاراً في المتصفح.

**الفرق:** OpenClaw يُرسل لك في واتساب على هاتفك دون أن تفتح المتصفح. KimiClaw يتطلب أن تكون في المتصفح أو لديك إشعارات المتصفح مُفعّلة.

## أبرز حالات الاستخدام الفعلية

### OpenClaw يتفوق في:

**1. الوكيل البريدي الكامل**
```
أنت: "راقب بريدي الوارد ولو جاءت رسالة من أي عميل جديد، أرد عليه تلقائياً بالرسالة الترحيبية وأضفه لـCRM"
```
يعمل باستمرار في الخلفية، لا تحتاج أن تفتح أي شيء.

**2. تتبع العملاء متعدد المنصات**
```
أنت: "لو ذُكر اسم شركتي في تويتر أو Reddit أو أي موقع تقييم، أخبرني فوراً في واتساب"
```

**3. الاستضافة المحلية للبيانات الحساسة**
الشركات التي لا تستطيع رفع بياناتها لأي سحابة — OpenClaw هو الخيار الوحيد المناسب.

---

### KimiClaw يتفوق في:

**1. المهام الفورية بدون إعداد**
```
أنت: "حلل هذا الكود وأخبرني بأي أخطاء محتملة"
[ترفع الملف مباشرة]
KimiClaw: [يُحلل ويرد خلال ثوانٍ]
```

**2. استخدام مهارات المجتمع الجاهزة**
تريد مهارة "تحليل PDF العقود" أو "ترجمة وتلخيص تقارير" — ابحث عنها في ClawHub وفعّلها بنقرة.

**3. التعاون الفوري**
يمكنك مشاركة جلسة مع زميل — كلاكما يرى نفس الوكيل ونفس المحادثة.

## المنافسون الجدد: PicoClaw وJoonClaw وما بعد

### PicoClaw — للأجهزة الضعيفة

مُصمَّم للعمل على أجهزة ARM الرخيصة (Raspberry Pi وOrange Pi وأجهزة بـ$10-30). النموذج المستخدم: Phi-3.5 Mini المُقطَّر — أخف بكثير لكن أقل قدرة.

**المستهدف:** المبتدئون الذين يريدون تجربة OpenClaw على أجهزة منخفضة التكلفة.

### JoonClaw (قريباً)

مشروع صيني قادم من Joon AI — يدّعي منشؤوه أن نموذجه المدمج المُحسَّن للمهام الوكيلية يتفوق على Kimi K2.5. لم يصدر رسمياً حتى أبريل 2026.

### NanoClaw — الأمان أولاً

نسخة معدَّلة من OpenClaw تعمل داخل **Docker container معزول** بصلاحيات صارمة — مصمم للشركات التي تحتاج أعلى مستويات الأمان ومنع تسريب البيانات.

## الاختيار النهائي: جدول القرار

**اختر OpenClaw إذا:**
- ✅ تُعطي الخصوصية أولوية قصوى
- ✅ تريد استخدام واتساب/تليجرام للتفاعل (لا متصفح)
- ✅ تريد بناء مهارات مخصصة لاحتياجاتك بالضبط
- ✅ تريد أتمتة مهام تعمل بالخلفية دون حضورك
- ✅ تريد استخدام أي نموذج تختاره (Claude/GPT/محلي)
- ✅ لديك خبرة تقنية أساسية (Node.js، سطر أوامر)

**اختر KimiClaw إذا:**
- ✅ تريد البدء فوراً دون أي إعداد تقني
- ✅ أولويتك السهولة وسرعة الإنجاز
- ✅ تريد الاستفادة من 5000+ مهارة جاهزة من المجتمع
- ✅ لا يوجد لديك خوادم أو حواسيب دائمة التشغيل
- ✅ بياناتك ليست بالغة الحساسية
- ✅ تريد التعاون مع زملاء في نفس الوكيل

**استخدم الاثنين إذا:**
- OpenClaw للمهام الحساسة والمؤتمتة الدائمة
- KimiClaw للمهام العرضية السريعة والتجريبية

## الخلاصة: عام "الوكيل الشخصي" بدأ

**OpenClaw** أثبت أن الوكيل الذكي الشخصي يمكن أن يكون مجانياً وخاصاً وقوياً — لكنه يتطلب بعض الجهد التقني.

**KimiClaw** أثبت أن نفس القدرات يمكن تقديمها لأي شخص بدون خبرة تقنية — لكن ببيانات على السحابة وتحكم أقل.

**الحقيقة الأعمق:** كلاهما يُشير لعالم قادم حيث لكل إنسان وكيل AI شخصي يعمل له 24 ساعة — يُنجز المهام الروتينية ويُتيح له التركيز على ما يستحق.

**هذا العالم ليس قادماً — بدأ.**
