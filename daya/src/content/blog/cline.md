## مقدمة: عندما أصبح لديك مبرمج آلي يعمل بدلاً منك

**Cline** هو وكيل برمجة مستقل (Autonomous Coding Agent) يعمل داخل VS Code. الفرق بينه وبين المساعدات الأخرى؟ **Cline لا يساعدك فقط — بل يعمل بشكل مستقل** لإنجاز المهام البرمجية كاملة.

**682 مليار token** — رابع أكثر وكيل برمجة استخداماً.

---

## ما هو Cline؟

Cline هو **إضافة مفتوحة المصدر لـ VS Code** تحول المحرر إلى بيئة عمل ذكية. يمكنك إعطاؤه مهمة معقدة وCline سينفذها خطوة بخطوة:

1. **يقرأ المشروع** — يفهم البنية والملفات
2. **يخطط** — يضع خطة تنفيذ
3. **ينفذ** — يكتب الكود، يختبره، يصحح الأخطاء
4. **يبلغ** — يخبرك بالنتيجة

### الفرق بين Cline وKilo Code:

| الميزة | Kilo Code | Cline |
|--------|-----------|-------|
| العمل المستقل | ❌ يحتاج توجيه | ✅ يعمل بشكل مستقل |
| كتابة الكود | ✅ | ✅ |
| الاختبار | محدود | ✅ متقدم |
| التصحيح | ✅ | ✅ متقدم |
| السعر | مجاني | مجاني |

## كيف تبدأ مع Cline؟

### التثبيت:

1. افتح VS Code
2. اذهب إلى Extensions
3. ابحث عن "Cline"
4. اضغط Install

### الإعداد:

```json
{
  "cline.model": "claude-3.5-sonnet",
  "cline.apiKey": "your-api-key",
  "cline.autoApprove": false
}
```

### الاستخدام:

```
1. افتح Cline من الشريط الجانبي
```
